Über das Programm

Der Hauptzweck dieser Web-App besteht im schnellen Anzeigen der OGD Wien Daten als Overlay über der OpenStreetMap. Bei einer großen Menge an Punkten werden diese zu einem einzelnen größeren Punkt zusammengefasst. Die Liste dieser Datensätze wird automatisch aus allen aktuell tatsächlich vorhandenen Datenlayern generiert und repräsentiert immer den aktuellen Stand.

Matching

Falls Punktdaten visualisiert werden besteht die Möglichkeit, diese automatisch mit OpenStreetMap Daten zu vergleichen. Relationen werden (noch) nicht beachtet. Die Geschwindigkeit des Matchings ist limitiert durch die Geschwindigkeit des Servers, welcher in diesem Fall ein sehr langsamer Raspberry PI ist.

Willkommen!

Diese Web-Applikation listet alle von der Stadt Wien als OpenGovernmentData freigegebenen Geodatensätze und zeigt diese bei Bedarf auf einer Karte an (mit Ausnahme des Geländemodells). Die Liste dieser Datensätze wird automatisch aus allen aktuell tatsächlich vorhandenen Datenlayern generiert. Auch die einzelnen Datensätze werden erst bei Aktivierung aktiv vom OGD Vienna Server geladen und stellen daher die aktuellste Version dar.

Performance

Wenn große Datensätze geladen werden kann die Bedienung träge werden.

Matching

Falls ein Datensatz des Types "Point" ausgewählt wurde, kann ein Vergleich mit der OpenStreetMap Datenbank durchgeführt werden.
